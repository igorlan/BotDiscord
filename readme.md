### Features

- Musicas
---
**Comandos paras musicas**

O comando !play pode receber o nome da musica + artista ou genero

```
!play nome_musica
```

!skip troca a musica
```
!skip
```

!volume aumenta o volume ou a baixa

```
!volume 30
```

Lista as musicas adicionadas
```
!playlist
```

Para de tocar as musicas
```
!stop
```

Pausa a musicas
```
!pausar
```

Retoma  a musicas
```
!retomar
```
